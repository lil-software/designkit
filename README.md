# DesignKit

[kit.design](https://kit.design)

A declarative syntax for the web. No frameworks, just plug and play.

It's simple:

**CSS**

```
<link href="https://kit.design/dist/main.0.0.1.css" rel="stylesheet" />
```

**HTML**

```
<vstack padding>
  <text font="large-title">Hello, World!</text>
</vstack>
```
